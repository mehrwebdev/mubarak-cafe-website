# Mubarak Cafe Website

## Overview
Mubarak Cafe is a modern, responsive restaurant website built to demonstrate a clean and structured frontend implementation for a food business.

The project focuses on layout clarity, visual hierarchy, and responsive behavior across devices.

## Features
- Responsive layout (desktop, tablet, mobile)
- Hero landing section
- Menu presentation
- Visit / contact section
- Organized folder structure

## Built With
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript

## Development Approach
The layout was structured using semantic HTML.  
Styling was built with a focus on spacing, alignment, and responsiveness.  
JavaScript was used to handle interactive elements where needed.

## Project Goal
To create a realistic restaurant website interface that reflects modern frontend standards and clean UI practices.
